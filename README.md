# Generating LLM data

Requirements:

- [homebrew](https://brew.sh/)
- [git](https://git-scm.com/)
- [ollama](https://ollama.com/)

## Getting started

Step 1. Install homebrew

Step 2. Install python

brew install python

Step 3. Install git

brew install git

Step 4. Clone git repo

git clone https://github.com/gumbelino/drillm

Follow instructions

Install git
Install python
Install pipenv
